### Prérequis

- Nodejs

### Installation

Installation du bot

1. Cloner le github

```
git clone https://github.com/andronedev/megaweb-bot.git
```

2. Installation des dépendances

```
npm i
```

Demarrage du bot :

```
node index.js
```
